Simplified MT940 Validator

Welcome to the MT940 Validator. This AngularJS application aims at performing a few validation steps in a simplified version of the MT940 file layout. This application was created by Fabio Fonseca for Rabobank.

Instructions

Click here to begin the validation process.

First step is to choose a CSV or XML file to be validated. Then click Validate! button and a validation report will be shown. Just that simple.
